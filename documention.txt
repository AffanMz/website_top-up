1. Pull Repo Terbaru (git pull origin dev).
2. migrate database (php artisan migrate:fresh).
3. Buka Web sqliteonline.com 'buatngecek databasenya'.
4. Di sqliteonline.com ke menu File -> Open DB -> Buka File project -> Folder Database -> pilih file database.sqlite.
5. cek semua table.
6. Folder Routes -> File Web.php
    Penjelasan :
    Setiap fitur CRUD Sudah Punya Halaman (tampil data,edit data,hapus data),
    view nya untuk tampil data pada tabel (itemspage, gamespage, orderspage, userspage),
    untuk view create dan edit data ada di folder masing.
